專案概述
================================================================================

一個完全基於 HTML5 + CSS3 + ES6 JavaScript 的 Rainbow Six Siege 戰術選角助手。
無任何依賴或打包工具，開箱即用，適合直接部署到任何 HTTP 伺服器。

特點：
- 純前端實現，無後端依賴
- 深色軍事風格，沉浸式遊戲體驗
- 完全響應式設計
- 三步驟直覺流程
- 244 個戰術組合資料庫

使用指南
================================================================================

【快速開始】
1. 進入專案目錄：
   cd /sessions/loving-sleepy-tesla/mnt/assets/r6-assistant

2. 啟動伺服器：
   python3 -m http.server 8000

3. 開啟瀏覽器：
   http://localhost:8000

【使用流程】
1. 在搜尋框輸入地圖名稱（中英文或別名）
2. 點擊地圖卡片選擇（橙色邊框表示選中）
3. 選擇攻擊或防守陣營
4. 從點位清單選擇特定點位
5. 查看推薦幹員、任務和技巧

【自訂選項】
- 修改顏色：編輯 style.css（搜尋 #f4a20a）
- 新增地圖：編輯 data/maps.json
- 新增幹員：編輯 data/operators.json
- 新增戰術：編輯 data/tactics.json

【部署方式】
- Python HTTP Server：python3 -m http.server 8000
- Node.js：npx http-server
- PHP：php -S localhost:8000
- 生產環境：任何支援靜態文件的伺服器（nginx、Apache 等）

================================================================================
已知限制與注意事項
================================================================================

【技術限制】
⚠️ 必須使用 HTTP 伺服器
   直接打開 HTML 會因 CORS 限制導致 JSON 加載失敗

⚠️ 需要啟用 JavaScript
   應用完全依賴 JavaScript 運行

⚠️ 網路連線要求
   Google Fonts 需要網路下載（約 50KB）

⚠️ 瀏覽器支援
   IE 11 不支援（使用 ES6 特性）
   建議使用最新版本的現代瀏覽器

【功能限制】
⚠️ 無本地儲存
   無收藏或歷史記錄功能（可自行擴展）

⚠️ 無離線模式
   無 ServiceWorker 支援

⚠️ 無即時更新
   資料需手動編輯 JSON 檔案
